# stock-chart-analysis

金融技术分析图生成器 —— 缠论 + 道氏理论 + 筹码分布 + 综合K线 + 竖版信息图。

## 触发条件

当用户提及以下关键词时激活：
- 分析股票 / 分析指数 / K线分析 / 技术分析 / 股票走势 / 分析行情 / A股分析
- 缠论 / 缠论分析 / 缠论分型 / 笔 / 中枢 / 背驰
- 道氏理论 / 波峰波谷 / 趋势线 / 支撑压力
- 筹码分布 / 筹码分析 / CYQ / 获利盘 / 主力成本 / 筹码集中度 / 90%成本
- 上证指数 / 深证成指 / 创业板指
- stock analysis / technical analysis / K-line / chart

## 执行参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `symbol` | string | `sh000001` | 股票/指数代码，支持 `sh000001`、`000001.SH`、`000001` |
| `name` | string | `上证指数` | 中文名称，用于图表标题 |
| `days` | int | `60` | 回溯交易日天数（与 `--full-history` 互斥） |
| `full-history` | flag | — | 拉取从上市日至今的全历史数据（覆盖 `--days`，推荐用于三档战法） |
| `output_dir` | string | `/tmp/stock_charts/` | 图表输出目录 |
| `upload` | bool | `true` | 生成图表后自动上传 COS |
| `no-upload` | flag | — | 跳过 COS 上传 |
| `no-charts` | flag | — | 仅输出 JSON 报告，不生成图表 |

## 执行流程

```
1. python generate_analysis.py --symbol sh000001 --name 上证指数 --days 60
   python generate_analysis.py --symbol sh000001 --name 上证指数 --full-history
   -> 拉取日K数据（akshare，兼容 sh/sz/.SH/.SZ/纯数字格式）
   -> 计算 MACD / MA 指标
   -> 缠论分析（分型、笔、中枢、背驰检测）
   -> 道氏理论分析（波峰波谷、趋势线、支撑压力位）
   -> 三档战法分析（一档/二档/三档趋势线、下跌三档、潜龙信号）
   -> MACD 金叉/死叉检测
   -> MA 均线排列判断（多头/空头/纠缠）
   -> 量价分析（放量上涨/缩量下跌等）
   -> CYQ 筹码分布分析（获利盘比例、平均成本、筹码峰值、集中度、主力成本区）
   -> 输出 JSON 分析报告到 stdout
   -> 生成缠论分析图 PNG
   -> 生成道氏理论分析图 PNG（含三档趋势线标注）
   -> 生成综合K线图 PNG（mplfinance 优先 + matplotlib fallback）
   -> 生成筹码分布图 PNG（通达信风格，K线+右侧筹码分布）
   -> 生成竖版信息图 PNG（1080x1920）
   -> 自动上传所有图表到 COS，返回公开 URL
```

## 输出

### 0. 图表交付规则
所有生成的图表必须以内联预览形式呈现（MEDIA 标签或图片嵌入），**禁止输出下载链接或 URL 表格**。用户只需要看图表，不需要点链接去下载。

### 1. 文字分析报告（JSON）
- `meta`: 股票代码、名称、分析日期、数据范围
- `chanlun`: 顶分型/底分型列表、笔列表、中枢列表、走势判断、背驰信号
- `dow`: 主要趋势、次要趋势、支撑位、压力位、波峰波谷、趋势线、量价关系
- `three_gears`: 三档战法分析结果
  - `up_gears`: 上涨一档/二档/三档（每档含 points①②③、角度、状态）
  - `down_gears`: 下跌一档/二档/三档
  - `hidden_dragon_signal`: 潜龙买点信号（下跌三档急跌+底分型）
  - `current_position`: 当前所处档位阶段
  - `current_point`: 当前点位编号（3=第三介入点）
  - `recommendation`: 三档战法专属操作建议
- `macd`: DIF/DEA/HIST 数值、金叉死叉信号、多空趋势
- `ma`: MA5/MA10/MA20 数值、均线排列（多头/空头/纠缠）
- `cyq`: 获利盘比例、平均成本、筹码峰值价格、90%/70%筹码集中度、主力成本区（低/高）、90%价格区间（低/高）、数据来源标记（`source: westock/local`）
- `recommendation`: 综合判断与操作建议

### 2. 缠论分析图
- K线蜡烛图（红涨绿跌）
- 顶底分型标注（顶分型▼、底分型▲）
- 笔（连接分型的折线）
- 中枢区间（半透明紫色矩形）
- 背驰标注（文字+箭头）
- MACD 副图（DIF/DEA/HIST）
- 成交量副图
- 深色背景 `#1a1a2e`，DPI 150

### 3. 道氏理论分析图（含三档战法）
- K线蜡烛图（红涨绿跌），全历史模式下自动裁剪显示最后600根K线
- 波峰波谷标注（波峰★、波谷★）
- 上升/下降趋势线（虚线）
- **三档趋势线**：一档线（绿色）、二档线（金色）、三档线（红色）
- **下跌三档趋势线**：虚线表示
- **第三点标注**：用特殊标记（◆/▼）+ 价格标签
- **潜龙信号**：下跌三档急跌后出现底分型时，用绿色高亮框标注
- 支撑压力位水平虚线
- 成交量副图
- 深色背景 `#1a1a2e`，DPI 150

### 4. 综合K线图（mplfinance 优先）
- 专业K线蜡烛图
- MA5 / MA10 / MA20 均线叠加
- 成交量柱状图
- MACD 副图（DIF/DEA/柱状）
- 支撑压力位水平虚线标注
- 分型标记
- 深色背景 `#0a0e17`，DPI 150
- 若未安装 mplfinance，自动 fallback 到 matplotlib 绘制

### 5. 筹码分布图
- 左侧 K线走势（含关键价位水平线：现价、平均成本、筹码峰值、主力成本区）
- 右侧筹码分布水平条形图（红涨绿跌配色，当前价以上红色/以下绿色）
- 底部成交量副图
- 右下角筹码指标文字面板（获利盘%、平均成本、筹码峰值、集中度、主力成本区）
- 深色背景 `#1a1a2e`，DPI 150
- 输出命名：`cyq_{market}{code}_{timestamp}.png`

### 6. 竖版信息图（1080x1920）
- 移动端适配竖版布局
- 大号当前价格与涨跌幅
- 迷你60日K线走势
- 分析卡片：缠论、道氏理论、支撑、压力、MACD、筹码分析
- 操作策略摘要
- 生成时间与数据来源

### 7. 图表交付规范
- 图表生成后自动上传至 COS 获得公网 URL
- **关键规则：禁止给出下载链接/URL**
- 所有图表必须以内联预览图形式输出：
  - 使用 `MEDIA:<url>` 标签嵌入预览（消息末尾单独一行）
  - 或 `![描述](url)` 图片语法直接渲染
- 文字分析报告中可引用图表内容辅助说明，但不再附 URL 表格
- 优先上传竖版信息图（infographic）作为主预览图，因其移动端友好且信息密度高
- 上传优先使用 cos_util（走自定义域名 nickstorage.top），fallback 到 coscli / Python COS SDK

## CYQ 筹码数据策略

CYQ 数据采用**混合来源**方案，优先外部接口，本地计算补充：

| 数据 | 来源 | 说明 |
|------|------|------|
| 获利盘比例 (`profit_ratio`) | 腾讯接口优先 | `fetch_cyq_westock()` 获取真实市场数据 |
| 平均成本 (`avg_cost`) | 腾讯接口优先 | 同上 |
| 90%集中度 (`concentration`) | 腾讯接口优先 | 同上 |
| 70%集中度 (`concentration_70`) | 仅腾讯接口 | 本地不计算此指标 |
| 密度曲线 (`price_bins`, `chip_distribution`) | 本地计算 | 用于筹码分布图可视化 |
| 筹码峰值 (`peak_price`) | 本地计算 | 三角分布模型峰值 |
| 主力成本区 (`main_force_low/high`) | 本地计算 | 最密集20%筹码区间 |
| 90%价格区间 (`price_low_90/high_90`) | 本地计算 | 90%筹码价格范围 |

**Fallback 规则**：腾讯接口不可用时，所有指标均由本地 `calculate_cyq()` 三角分布模型计算，`source` 字段标记为 `local`。
